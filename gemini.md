# Konstytucja Projektu (Project Constitution) - WallDecor

## Cel Główny (North Star)
Zapewnienie architektom pełnej przejrzystości rozliczeń i wzmocnienie lojalności poprzez system Cashback (Saldo Partnerskie), minimalizując jednocześnie obciążenie operacyjne zespołu WallDecor.

## Kluczowe Parametry (Business Logic)
- **Minimum do wypłaty:** 300 PLN netto.
- **Cashback:** 2% od wartości netto produktów (wyłączając montaż).
- **Ważność Cashback:** 12 miesięcy (FIFO).
- **RBAC:** ARCHI (self-service), STAFF (ops), ADMIN (full control).

## Integracje (Integrations)
- **Framework:** Next.js (App Router)
- **Autoryzacja:** NextAuth.js (Database Strategy)
- **Baza Danych:** MySQL (cPanel/Localhost)
- **Konektywność:** Gemini API (do analizy danych finansowych)

## Slogan Projektu
*Transparentność buduje zaufanie, lojalność buduje przyszłość.*
